JsCalcTdd - TDD proof of concept project for minimalist TDD for Js
==================================================================
This is a proof of concept that shows how to write unit tests for
Javascript when are not allowed to download tools like JsUnit.

Its useful because you can automatically run the tests
with the usual JUnit test harness tools.

It requires java 6 or higher.

For the sake of simplicity it's an eclipse project. Automatic build
tools are not supported yet.
